Example of practical work (modified for copyright):


- Question 1: On the Wazuh log file identify all events that indicate potential credential compromise. Check event ID 4668.

First step: Get the file and unzip it.

wget [https://pathof.thefile.com/wazuh_export.zip](https://pathof.thefile.com/wazuh_export.zip)
unzip wazuh_export.zip

Second step: Use grep to find the parent process name.

grep -B5 "mimikatz" wazuh_export.json
"newProcessName": "C:\\\\Users\\\\Administrator\\\\Downlads\\\\mimikatz.exe",
"parentProcessName": "C:\\\\Program Files\\\\Mozilla Firefox\\\\firefox.exe",

Answer: "newProcessName": "C:\\\\Users\\\\Administrator\\\\Downlads\\\\mimikatz.exe"


- Question 2: Identify events that show persistence mechanism. Find the imagePath for persistence on host DB01.

First step: Search using jq to filter the JSON file.

jq '.[] | select(.source.agent.name == "DB01" | select(._source.rule.groups[] == "persistence")' wazuh_export.json
"_source": {
  "rule": {
    "description": "New Windows service installed",
    "groups": ["service", "persistence"]
  },
  "data": {
    "win": {
      "eventdata": {
        "serviceName": "PSEXESVC",
        "imagePath": "C:\\\\Windows\\\\PSEXESVC.exe"
      }
    }
  }
}

Answer: "imagePath": "C:\\\\Windows\\\\PSEXESVC.exe"


- Question 3: Identify exfiltration activity. What is the external IP address to which diagnostics_data.zip was uploaded?

grep -B5 "diagnostics_data.zip" wazuh_export.json

Answer: "destinationIp": "93.184.216.34"
